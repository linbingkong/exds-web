研究分析工作台，区别于「月内交易行情」（盯盘工作台）。提供历史行情分析、价格发现路径研究、商机扫描和收益回测能力。

- 菜单组：交易策略
- 菜单名：月内行情分析

## 三、六大分析模式

### 模式 1：单标的全周期复盘

选择任一 `cj_time`（已交割/未交割），剖析其完整生命周期。

```
┌─ 时间带 ──────────────────────────────────────────────────┐
│  D+11 ─ D+10 ─ ... ─ D+2  ─ 交割  ─ 现货实际             │
│   ●─────●──────────●──────●──────────────●               │
└──────────────────────────────────────────────────────────┘

┌─ 主图（叠加比较） ─────────────────────────────────────────┐
│  ─── 滚动撮合均价路径（撮合周期内全部轮次）                 │
│  ─── 现货实际出清价（已交割时显示，作为"答案"）             │
│  ─── 日前申报价（参考）                                    │
│  时段多选：默认按尖峰平谷各取一段                           │
└──────────────────────────────────────────────────────────┘

┌─ 关键指标卡 ──────────────────────────────────────────────┐
│ 首日均价 ¥302  末日均价 ¥318  现货均价 ¥325  偏差 -2.2%  │
│ 价格波动率  累计成交  价格收敛度  与现货相关系数            │
└──────────────────────────────────────────────────────────┘
```

已交割标的的额外能力：把现货实际价作为"事后答案"叠加，立刻能看到滚动撮合是否系统性偏离现货。

### 模式 2：时段横向对比

固定一个时段（如 period 32 = 15:30-16:00），观察该时段在多个交割日的表现。

- X：交割日（连续 N 天）
- Y：该时段最终成交均价 vs 现货实际价
- 多维筛选：按工作日/周末、按月、按节假日类型
- 应用：识别"周一晚高峰系统性偏低"等结构性现象

### 模式 3：模式聚类（热力图全景）

```
            时段→  1   2   3  ...  32  33  ...  48
   交割日↓
   05-08         蓝  蓝  蓝       红  红       浅
   05-07         蓝  浅  蓝       红  红       浅
   05-06         蓝  蓝  浅       橙  红       浅
   ...
```

- 颜色：成交均价
- 一眼看出：哪些时段长期偏贵、哪天集体异动、节假日是否漂移
- 支持切换"色阶维度"：均价 / 净挂量 / 与现货偏差

### 模式 4：价格发现路径形态学

跨多个已交割标的，把每个 `cj_time` 的价格轨迹标准化：

- X：距交割天数（从 11 → 0）
- Y：价格相对最终现货价的比例（`price / spot - 1`）
- 每条线是一个交割日的轨迹
- 叠加显示：均值带 + 95% 分位带

回答的核心问题：滚动撮合从 D+11 到交割，平均偏离现货多少？什么时候开始收敛？是否存在"D+5 之后系统性贴水"这样的规律？

可分组查看：工作日 vs 周末 / 月初 vs 月末 / 高负荷周 vs 低负荷周。

### 模式 5：商机扫描器

针对当前在交易中的所有标的+时段组合，跑规则引擎，输出排行榜：

| 标的 | 时段 | 信号 | 强度 | 解释 |
|------|------|------|------|------|
| D+3 | 32 | 历史分位极低 | ★★★★ | 当前 ¥285 处于历史 6% 分位，5 天均值 ¥318 |
| D+5 | 36 | 净挂量异常 | ★★★☆ | 买挂/卖挂 = 4.2，远高于正常 |
| D+8 | 18 | 偏离形态均值 | ★★★☆ | 相对历史 D+8 路径低 8.3% |
| D+2 | 40 | 临近交割加速 | ★★☆☆ | 最近 3 轮连续上涨累计 +5.6% |

每条信号点击展开详情，同时关联跳转到模式 1 的单标复盘。

### 模式 6：收益回测

对已交割标的做事后回测，验证策略：

输入：
- 标的范围（最近 N 天交割的全部标的）
- 策略：早买（D+11 首轮）/ 晚买（D+3 最后一轮）/ 均匀分批 / 信号触发买入
- 持仓口径：哪些时段 / 全时段

输出：
- 累计盈亏（撮合价 vs 现货实际价）
- 胜率
- 单时段盈亏分布
- 按时段类型分组的表现（峰平谷各自盈亏）
- 与基准比较（如全时段均匀买入）

意义：把"商机直觉"变成"经过历史验证的可执行策略"。

## 四、跨页面通用约定

1. 配色统一：红涨绿跌；峰平谷固定色谱（峰=橙红、平=灰、谷=蓝、深谷=深蓝、尖峰=深红）。
2. 时段类型展示：所有表格、图表的横轴时段都带类型色标。
3. 快捷跳转：从模式 5 的扫描结果可一键进模式 1；从月内交易行情的"分位极端"列也能直跳模式 1。
4. 导出：表格右上角支持导出 CSV/Excel；图表支持导出 PNG。
