# 交易总览首页实施计划

## 1. 背景与目标

电力交易辅助决策系统（exds-web）当前 `/dashboard` 路由指向空白 `PlaceholderPage`。本计划构建一个专业大屏首页，作为系统登录后的默认着陆页，归纳展示核心运营数据。

**核心要求**：
- 浅底高对比专业大屏，符合 MUI 标准业务风格
- 适配 **2560×1440 分辨率**，全屏无滚动
- 首页 Tab 标签**不可关闭**
- 九宫格布局：中间重点 + 左右各三格

## 3. 各面板详细设计

### ④ 结算收益面板（C位上方）— 最重要

**位置**：中列上半区，占中列 50% 高度。

#### KPI 指标行

单行 5 个指标卡，面板标题旁以小字体显示「数据截至 YYYY-MM-DD」（取 `daily_details` 最后一条有数据的日期）。

| 指标 | 格式 | 数据源 |
|------|------|--------|
| 年度累计毛利 | xx.xx 万元 | `/settlement/overview` 逐月累加（含当月日清预结算） |
| 本月毛利 | xx.xx 万元 | `/settlement/overview?month=本月` → `summary.gross_profit` |
| 购电均价 | xxx.xx 元/MWh | 同上 → `summary.wholesale_avg_price` |
| 售电均价 | xxx.xx 元/MWh | 同上 → `summary.retail_avg_price` |
| 批零价差 | ±xx.xx 元/MWh | 同上 → `summary.price_spread`，正值绿色负值红色 |

> **年度累计毛利计算口径**：1月~上月的已结算毛利 + 本月截至最新日清的预结算毛利，三者累加。「截至日期」标注使数据时效性透明。

#### 主图表

**本月每日批零价差柱状图 + 累计均价差折线图**。

- 图表类型：`ComposedChart`（Recharts）
- 柱状图：`Bar` dataKey=`price_spread`，正值 `#4caf50` 绿、负值 `#f44336` 红
- 折线图：`Line` dataKey=`cumulative_avg_spread`，颜色 `#ff9800` 橙
- 双 Y 轴：左轴价差 (元/MWh)，右轴累计均价差
- X 轴：本月所有日期（含未来无数据日）
- 零线：`ReferenceLine y={0}`

**参考**：[PreSettlementOverviewPage.tsx](file:///d:/Gitworks/exds-web/frontend/src/pages/PreSettlementOverviewPage.tsx#L617-L671) 的 [renderRightChart](file:///d:/Gitworks/exds-web/frontend/src/pages/PreSettlementOverviewPage.tsx#613-673) price 模式。

### ① 客户概览面板（左上）

**位置**：左列第 1 格。

| 元素 | 展示形式 | 说明 |
|------|---------|------|
| 签约客户总数 | **大号数字**（如 `42`） | 当前活跃客户数量 |

- 签约电量：`retail_contracts` 的 `purchasing_electricity_quantity` sum，去年同口径对比

### ⑥ 总体负荷面板（右上）

**位置**：右列第 1 格。

| 元素 | 展示形式 | 说明 |
|------|---------|------|
| 模式切换 | **小型 ToggleButton**：月 / 日 / 曲线 | 三种视图切换 |
| 月模式 | 近 12 月柱状图（BarChart） | 月度用电量趋势 |

- 曲线：`GET /api/v1/total-load/curve?date=今天&compare_type=yesterday`
- 统计：`GET /api/v1/total-load/statistics?date=今天&scope=daily`

### ⑦ 市场价格面板（右中）

**位置**：右列第 2 格。

| 元素 | 展示形式 | 说明 |
|------|---------|------|
| 指标数字 | 两行：中长期均价 / 实时现货均价 | 近期平均值 |
| 折线图 | **近 30 天双折线**：蓝色=中长期合同均价，红色=实时现货均价 | 趋势对比 |
| 阴影区 | 两线之间填充半透明色 | 价差可视化 |

**数据源**：`GET /api/v1/contract-price-trend/price-trend?start_date=30天前&end_date=今天&spot_type=real_time`

返回 `daily_trends[]`，每条包含：
- `contract_vwap`：中长期合同日均价
- `spot_vwap`：现货日均价（实时）
- `vwap_spread`：价差

指标数字取 `spread_stats.avgSpread`（平均价差）。

**图表参考**：[MarketDashboardTab.tsx](file:///d:/Gitworks/exds-web/frontend/src/components/MarketDashboardTab.tsx#L427-L641) 的 [PriceChart](file:///d:/Gitworks/exds-web/frontend/src/components/MarketDashboardTab.tsx#426-642) 双折线样式（实时红 `#f44336` / 日前蓝 `#2196f3`），但粒度为每日数据点而非 96 时段点。

| 元素 | 展示形式 | 说明 |
|------|---------|------|

## 4. 后端 API 设计

### [NEW] [v1_dashboard.py](file:///d:/Gitworks/exds-web/webapp/api/v1_dashboard.py)

新建 Dashboard 聚合路由，所有接口为**只读 GET**，不需要写权限。

| 接口路径 | 说明 | 复用逻辑 |
|---------|------|---------|
| `GET /dashboard/settlement-kpi` | 年度累计+本月结算KPI+截至日期 | 复用 `SettlementService` |
| `GET /dashboard/settlement-chart` | 本月每日价差+累计均价差 | 复用 `/settlement/overview` 的 `daily_details` |
| `GET /dashboard/trade-summary` | 本月四品种交易概览 | 复用 `MonthlyTradeReviewService` |
| `GET /dashboard/customer-overview` | 客户数+合同分布+签约电量 | 直接查 `customer_archives` + `retail_contracts` |
| `GET /dashboard/customer-profit-contribution` | 客户收益贡献(正/负) | 复用 `CustomerProfitAnalysisService` |
| `GET /dashboard/customer-load-ranking` | TOP5客户电量占比 | 新增聚合 `load_data` |
| `GET /dashboard/alerts` | 最近告警列表 | 查 `system_logs` |

## 6. 视觉设计规范

| 项目 | 规范 |
|------|------|
| 背景色 | 沿用 theme `background.default: '#f4f6f8'` |
| 面板容器 | MUI `Paper variant="outlined"`，白底 `borderRadius: 8` |
| KPI 数字 | 大号字体 `variant="h5"`，正值 `#4caf50` 绿、负值 `#f44336` 红 |
| 图表颜色 | 主色 `#1976d2` 蓝系，正负语义色保持一致 |
| Recharts 焦点 | 按项目规范，统一 `& .recharts-surface:focus { outline: none }` |
| 字体 | 沿用系统默认，面板标题 `fontWeight: 800` |
